üí° Usage Guide

  1. Enter the text content you want to convert to image in the text input box
  2. Select font family, adjust font size and weight
  3. Choose text color and background color using the color pickers
  4. Set canvas size (width and height) for your image
  5. Choose text alignment (left, center, or right)
  6. Adjust padding to control spacing around text
  7. Enable optional effects: shadows, borders, or gradient backgrounds
  8. Click "Generate Image" to create your image
  9. Preview the result and download in PNG or JPG format
