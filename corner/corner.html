üìã Feature Description

Our Rounded Corners tool provides powerful capabilities to add rounded corners to your images:

  • Multiple Corner Types: Support uniform corners, custom corners, and circular images
  • Precise Control: Adjust corner radius from 0 to 100 pixels with real-time preview
  • Quick Presets: One-click access to common corner sizes (small, medium, large, extra large)
  • Background Options: Transparent background, solid color, or gradient background
  • Format Support: Export in PNG, JPEG, or WebP formats with quality control

üí° Usage Guide

  1. Click to select an image or drag & drop it to upload
  2. Choose corner type: Uniform Corners, Custom Corners, or Circular Image
  3. For uniform corners, adjust the corner radius slider or use quick preset buttons
  4. For custom corners, adjust each corner's radius individually (top-left, top-right, bottom-left, bottom-right)
  5. For circular images, set the circle size and enable auto-crop to square if needed
  6. Choose background type: Transparent, Solid Color, or Gradient
  7. Select background color if using solid color or gradient
  8. Choose size mode: Keep Original Size or Custom Size
  9. Select output format (PNG, JPEG, or WebP) and quality settings
  10. Click "Preview Effect" to preview, then "Process Image" to generate
  11. Download your rounded corner image
